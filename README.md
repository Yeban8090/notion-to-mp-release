# Notion to MP - Release
一款帮助你将 Notion 文章一键排版复制到微信公众号的浏览器插件。

## 下载
- notion-to-mp.crx：直接安装版本

## 功能特点

- 一键复制 Notion 文章到微信公众号
- 支持多种主题样式切换
- 支持自定义字体和字号
- 保持 Notion 文章的格式和样式
- 优化代码块显示效果
- 完美支持图片、表格等元素

## 支持的样式

- 标题（H1-H4）
- 段落文本
- 引用块
- 代码块（带语法高亮）
- 列表（有序和无序）
- 表格
- 图片
- 分割线

## 主题列表

- 默认主题
- 极简主题
- 优雅主题
- 暗黑主题
- 学术主题
- 叶帮主题
- 阿篱主题

## 使用方法

1. 在 Chrome 浏览器中安装插件
2. 打开任意 Notion 页面
3. 点击插件图标
4. 选择喜欢的主题
5. 点击复制按钮
6. 直接粘贴到微信公众号编辑器中
